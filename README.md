# 2026_NWAFU_undergraduate_latex_template
2026届西北农林科技大学本科生毕业论文latex模板

因为自己排版需要，在耿楠老师的latex模板基础上进行修改以适配最新版本的模板，技术不精可能存在问题的地方，本人不负任何责任。

主要改动包括：

1. 改动cls文件，建议直接下载放在自己的根目录下。

2. 新增KaiTi_GB2312.ttf字体文件，适配独创性声明页字体格式。

3. main.tex中begin{decument}之前新增封面重定义内容。

4. settings/package.tex文中新增两行关于字体配置信息。

5. 在main.tex文件中\mainmatter之前include新建的术语表、主要符号注释表等内容。
